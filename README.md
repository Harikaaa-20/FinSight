# FinSight – Financial Document Analysis

FinSight is an NLP-based system that extracts structured insights from unstructured financial documents using machine learning techniques. This project was developed as part of my Infosys Springboard internship.

## Tech Stack
- Python
- spaCy, BERT
- Node.js
- NLP pipelines

## Features
- Extracts key financial insights from raw text documents
- Processes large unstructured datasets efficiently
- Uses NLP models for information extraction and analysis
- Full-stack setup with backend processing and frontend interface

## My Contribution
- Built backend data processing pipelines using Python
- Integrated NLP models (BERT, spaCy) for text analysis
- Designed workflows to transform unstructured financial data into structured outputs
- Worked on improving processing efficiency and data handling
